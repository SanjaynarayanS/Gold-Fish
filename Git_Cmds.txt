--Prequisites in command prompt
git config --global user.name "type github user name"
git config --global user.email "type github email account"

--Cloning in VS Code 
>git clone https://github.com/SanjaynarayanS/Gold-Fish.git //clones the github repo in Local(Git cloned area and working folder)

--Understanding
Github - repository in Github Platform
Git Cloned Folder - cloned repo in Local
Working Folder - Copy of Git Cloned Folder

--Create a New Repo using Git Commands
>git init //initializes an empty repo
>git add . // all created//untracked/modified/deleted files in working folder to added(stagging area)
>git commit -m "added all new files"
Create an empty Repo in github 
>git remote add origin https://github.com/SanjaynarayanS/Project-Forget.git
>git branch => master (default branch name is master not main)
>git push origin master
         (or)
>git branch -M main (name change from master to main)
>git push origin main

--created a new file in Working folder 
>git status => untracked
>git add Git_Cmds.txt // untracked to added(stagging Area) (or)  git add .
>git Status => Changes to be commited
>git commit -m "Added Git_Cmds.txt"  // Working folder to Git cloned Folder
>git Status => use "git push" to publish your local commits
>git push origin main// Git cloned folder to Github

--modified an existing file in working folder 
>git status => Modified
>git add Git_Cmds.txt // untracked to added(stagging Area) (or)  git add .
>git Status => Changes to be commited
>git commit -m "Modified Git_Cmds.txt"  // Working folder to Git cloned Folder
>git Status => use "git push" to publish your local commits
>git push origin main// Git cloned folder to Github

--deleted an existing file in working folder 
>git status => deleted
>git add Git_Cmds.txt // untracked to added(stagging Area) (or)  git add .
>git Status => Changes to be commited
>git commit -m "Deleted Git_Cmds.txt"  // Working folder to Git cloned Folder
>git Status => use "git push" to publish your local commits
>git push origin main// Git cloned folder to Github

--Created a newfile in Github repo
>git pull // pulls the changes made in github repo to working folder
